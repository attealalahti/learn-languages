(inner) GET request(fromopt, toopt) → {Array.<Object>}

Attempts to get all word pairs from the database to send them to the requester. All sent word pairs' languages will be in the same order. If both from and to parameters are defined in the request, only word pairs from the specified languages will be returned.

Parameters:

Which language the words are supposed to be translated from.

Attempts to update a word pair in the database. Sends back an error if the given word pair object does not have the right kind of data in the right fields. Will only update the translations of the word, not which languages the words are in.
